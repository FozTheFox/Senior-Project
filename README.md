# Auto/Car Body Paint and Repair Service Management System & Mechanic Mobile App

A comprehensive digital transformation project developed for an auto/car body paint service in Roi Et, Thailand. This graduation project aims to streamline daily workflows, digitize documentation, and enhance operational efficiency for both management and mechanics.

## Key Features

### Admin Web Dashboard (Document & Data Management)
* **Document Digitization:** Transformed paper-based workflows into a secure digital system, effectively preventing the loss of critical work orders and customer documents.
* **Centralized Management:** A complete overview of ongoing repairs, parts inventory, and mechanic assignments.
* **Data Security & Integrity:** Designed robust data structures to ensure long-term data preservation and secure access.

### Mechanic Mobile Application (Workflow Optimization)
* **Task Management:** Clear, step-by-step digital workflows tailored specifically for auto body mechanics.
* **Workload Reduction:** Minimized administrative friction, allowing mechanics to focus entirely on repair tasks rather than paperwork.
* **Time Tracking:** Automated logging of working hours per task to evaluate performance and calculate wages accurately.
